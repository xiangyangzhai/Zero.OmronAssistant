# Zero.OmronAssistant
仓库地址
https://github.com/xiangyangzhai/Zero.OmronAssistant.git


使用说明
0. 导出Omron PLC全局变量，并保存为ANSI格式的txt文档
1. 【参数设置】-【打开全局变量文件】导入全局变量
2. 【主界面】-【开始读取】，等待读取完成
3. 【主界面】-【保存读取变量】，将上一步获取的变量值存储为json格式的txt文档
4. 【主界面】-【恢复PLC变量值】，选择上一步保存的文档，将恢复PLC对应的变量值，等待完成

重要提示
鉴于对PLC变量的直接操作可能引起误操作、撞机等等不可预知的后果，建议读写不涉及动作的数据，而不是一些触发动作的开关量，以及可能引发用户交互的报警相关内容。

以及由以上数据类型组成的数组和结构体

---
2023年5月13日
本软件主要解决现场调试时，新增全局变量需要重新下载程序，已经标定的部分全局变量的值会被清空。
通过本软件可以将全局变量的值保存，更新PLC程序后，将保存的全局变量的值重新写入到PLC即可。
